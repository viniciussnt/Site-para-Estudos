CSS3

O que é o CSS3?

O CSS3 é utilizado para estilizar as telas que desenvolvemos com o HTML5 possivilitando que elas sejam bonitas e funcionais.

Onde o CSS3 está presente?

Na grande maioria das páginas da web. Inclusive, é bem provável que você nunca tenha visto uma página sem CSS.

A sintaxe do CSS3

                h1 {
                    color: blue;
                    font-size: 12px;
                }

O h1 é o seletor. Seletores nada mais são do que o elemento HTML (ou uma classe a ser aplicar em um elemento HTML).

O color e o font-size são as propriedades. Elas são atributos do elemento HTML onde será aplicada uma regra.

O blue e o 12px são valores. Valores é uma característica específica a ser assumida pela porpriedade.

Selecionando elementos do HTML para estilizar

Podemos selecionar não somente as tags para estilizar mas também as classes e os ids.

Por tag: 

                body {
                    background-color: green;
                }

Por id: 

                <h1 class="title"> 
                  Hello <span id='highlight'>World</span>
                </h1>

                #highlight {
                    font-size: 22px;
                }

Por classe: 

                <h1 class="title"> 
                  Hello <span id='highlight'>World</span>
                </h1>

                .title {
                    color: green;
                }

Alguns estilos CSS

Color:

color: white
color: #efefef

Background-color:

background-color: grey
background-color: #efefef

Font-size

font-size: 14px
font-size: 2cm

CSS Box-Model

O box model no CSS é essencialmente uma caixa que envolve todos os elementos HTML.

Ele consite em margens, bordas, preenchimento e o conteudo real.

Exemplo de margin: 

                h1 {
                    margin-top: 100px;
                    margin-right: 100px;
                    margin-bottom: 150px;
                    margin-left: 80px;
                }

Exemplo de padding: 

                h1 {
                    padding-top: 50px;
                    padding-right: 30px;
                    padding-bottom: 50px;
                    padding-left: 80px;
                }

Adicionando CSS ao projeto

Inline

O CSS inline é uma forma de estilizar os elementos específicos dentro de um documento HTML.

Através do atributo style podemos palicar qualquer estilo CSS a um elemento de uma página.

                <h1 style="color: blue;">Olá, Mundo!</h1>

Atenção: Essa forma é a menos recomendada de se usar CSS para estilizar.

Interno

É uma outra forma usada que ao invés de ser usada diretamente na tag, é usada no head fazendo o uso da tag style.

                <head>
                    <style>
                      body {background-color: powderblue;}
                      h1 {color: blue;}
                      p {color: red;}
                    </style>
                  </head>

Externo

É a forma mais correta usando um arquivo .css externo

                index.html
                <head>
                    <link rel="stylesheet" href="style.css">
                  </head>

                  style.css
                  body {
                  background-color: powderblue;
                  }
                  h1 {
                  color: blue;
                  }
                  p {
                  color: red;
                  }

Pseudo Classes

Uma pseudo-classe é usada para definir um estado especial de um elemento

Por exemplo, ela pode ser usada para:

Estilizar um elemento qunado o usuário passar o mouse sobre ele. 

                a:hover {
                    color: #ff00ff;
                }

Estilizar os links visitados e não visitados de maneira diferente 

                a:visited {
                    color: #00ff00;
                }

Estilizar um elemento quando receber foco 

                a:active {
                    color: #0000ff;
                }

Variáveis em CSS

As variáveis CSS são entidades definidas por desenvolvedores, contendo valores espcíficos para serem reutilizdos ao longo do projeto.

Sáo configurads usando esta notação 

                --cor-principal: black;

São acessadas usando a função var(). 

                color: var(--cor-principal);

Exemplo prático 

                :root {
                    --main-bg-color: brown;
                  }
                  
                  div {
                    background-color: var(--main-bg-color);
