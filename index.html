HTML5

O que é o HTML 5?

É uma linguagem de marcação criada para estrutuar e apresentar um conteúdo na Web.

O HTML5 usa Tags para definir quais os conteúdos de uma página Web.

As Tags são anotações com a sequinte anatomia: 

                <p>[Conteudo]</p>
Mas também podem não ter um fechamento: 
                <img src="https://example.com/image.png/>"

Tags Fundamentais

Hierarquia de Títulos

São Tags usadas para a criação de títulos. 

                <h1>Esse é o meu título</h1>
As Tags de título vão de <h1> até <h6>

Parágrafo

Essa Tag mostra o texto como um parágrafo na página. 

                <p>Esse é um parágrafo</p>

Divs

Servem para envolver diversos elementos do código e assim podermos aplicar estilos à ele com CSS.
As Divs não apresentam nenhuma mudança visual na página. 

                <div>

                    <h1>Meu Título</h1>
                    <p>Meu parágrafo</p>

                </div>

Span

É semelhante à div, ele serve para envolver um ou mais elementos sem apresentar mudanças visuais (até usarmos CSS, a principal diferença para a div é que em geral os browsers pulam uma linha para serparar ps elementos de uma div e os demais. 

                <p>Meu <span>parágrafo</span> </p>

Lista não ordenada e ordenada

Usamos <ul> para criar uma lista não ordenada dos elementos. 

                <ul>
                    <li>Elemento 1</li>
                    <li>Elemento 2</li>
                </ul>

E para criar uma lista ordenada usamos <ol>. 

                <ol>
                    <li>Elemento 1</li>
                    <li>Elemento 2</li>
                <ol>

Nota: A tag <li> é usada para representar um item na lista

Links

As tags podem possuir alguns atributos extras como por exemplo <a href=""> que serve para criarmos um link entre duas página 

                <a href="https://google.com">Google</a>

Imagens

Para inserirmos uma imagem na página usarmos a tag <img src=""> 

                <img src:"https://example.com/image.png"/>

Anatomia de um documento HTML

            <!DOCTYPE html>
            <html lang="pt-br">
            <head>
                <meta charset="UTF-8">
                <title>Documento</title>
            </head>
            <body>
                [Conteúdo]
            </body>
            </html>

<!DOCTYPE html>

A DOCTYPE precisa ser declarada no inicio de todos as páginas, com ele nós informamos ao browser que o documento é uma página em HTML

<html lang="pt-br"

Essa tag engloba todas as outras tags do documento dando unidade a eles.

Ela tem que está sempre presente e é conhecida como tag root (raiz) do documento.

O atributo lang tem como objetivo informar ao browser a linguagem (portguês no caso) que foi utilizado no documento.
