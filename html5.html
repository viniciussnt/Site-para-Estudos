HTML5

O que é o HTML 5?

É uma linguagem de marcação criada para estrutuar e apresentar um conteúdo na Web.

O HTML5 usa Tags para definir quais os conteúdos de uma página Web.

As Tags são anotações com a sequinte anatomia: 

                <p>[Conteudo]</p>
Mas também podem não ter um fechamento: 
                <img src="https://example.com/image.png/>"

Tags Fundamentais

Hierarquia de Títulos

São Tags usadas para a criação de títulos. 

                <h1>Esse é o meu título</h1>
As Tags de título vão de <h1> até <h6>

Parágrafo

Essa Tag mostra o texto como um parágrafo na página. 

                <p>Esse é um parágrafo</p>

Divs

Servem para envolver diversos elementos do código e assim podermos aplicar estilos à ele com CSS.
As Divs não apresentam nenhuma mudança visual na página. 

                <div>

                    <h1>Meu Título</h1>
                    <p>Meu parágrafo</p>

                </div>

Span

É semelhante à div, ele serve para envolver um ou mais elementos sem apresentar mudanças visuais (até usarmos CSS, a principal diferença para a div é que em geral os browsers pulam uma linha para serparar ps elementos de uma div e os demais. 

                <p>Meu <span>parágrafo</span> </p>

Lista não ordenada e ordenada

Usamos <ul> para criar uma lista não ordenada dos elementos. 

                <ul>
                    <li>Elemento 1</li>
                    <li>Elemento 2</li>
                </ul>

E para criar uma lista ordenada usamos <ol>. 

                <ol>
                    <li>Elemento 1</li>
                    <li>Elemento 2</li>
                <ol>

Nota: A tag <li> é usada para representar um item na lista

Links

As tags podem possuir alguns atributos extras como por exemplo <a href=""> que serve para criarmos um link entre duas página 

                <a href="https://google.com">Google</a>

Imagens

Para inserirmos uma imagem na página usarmos a tag <img src=""> 

                <img src:"https://example.com/image.png"/>

Formulário

A tag <form> é especial porque ela nos permite submeter informações para um servidor.

Por exemplo, enviar dados pde um formulário para serem salvos em um banco de dados.

Exemplo: 

                <form action="https://google.com/search" method="get"
                <label for="q">Sua pesquisa:</label>
                <input type="text" name="q" required>

                <input type="submit" value="Buscar!">

Form

A principal tag de um formulário é o <form> e ela pode ter alguns atributos

<action> - Informa qual a URL do servidor onde vamos submeter as informações

<method> - Diz qual o método iremos usar.
get: geralmenteusado para pegar informações.
post: geralmente usado para grava informações no servidor.

Input

A tag input é a responsável por mostrar os campos de entrada e dados, essa tag também possui alguns atributos.

type - Diz qual o tipo de daos queremos receber, por exemplo texto, número, data e etc.

name - Para que o servidor consiga saber onde está cada informação importante que submetemos nós precisamos nomear cada um dos inputs.

required - Esse atributo diz ao browser que nós só podemos submeter esse form se esse campo for preenchido (caso tentemos fazer isso o browservai mostrar uma mensagem de erro).

Obs: o input que possui o type submit é sempre o botão de submissão do form.

Label

A tag label é um texto que geralmente acompanha o input e informa do que se trata aquele campo, ela possui pelo menos um atributo

for - Esse atributo aponta para o id do input que aquele label está associado

Anatomia de um documento HTML

            <!DOCTYPE html>
            <html lang="pt-br">
            <head>
                <meta charset="UTF-8">
                <title>Documento</title>
            </head>
            <body>
                [Conteúdo]
            </body>
            </html>

<!DOCTYPE html>

A DOCTYPE precisa ser declarada no inicio de todos as páginas, com ele nós informamos ao browser que o documento é uma página em HTML.

<html lang="pt-br"

Essa tag engloba todas as outras tags do documento dando unidade a eles.

Ela tem que está sempre presente e é conhecida como tag root (raiz) do documento.

O atributo lang tem como objetivo informar ao browser a linguagem (portguês no caso) que foi utilizado no documento.

<head>

A tag head envolve todas as outras tags que você deseja incluir no documento que não vão ser mostradas ap súmario.

Por exemplo uma descrição e um título da página que aparece no google, as fontes que serão baixadas, os estilos e etc.

<body>

A tag body envolve todas as tags que serão mostradas no browser como títulos, imagens, links, lista e etc.

Meta Tags

São tags que ficam dentro do gead e que não aparecem diretamente na página mas nos ajudam a dar informações sobre ela para o browser e os mecanismo de busca como o Google.

<meta charset="UTF-8">

A tag diz ao browser que vamos usar o padrão de encode UTF-8.

UTF-8 é um tipo de codificação binária para representar os caracteres.

<meta name="description" "content=""

Essa meta tag informa qual a descrição do conteúdo da página, geralmente ela pe usada pelo Google para mostrar do que a página se trata.

<meta name="author" content="">

Essa tag informa quem é o autor daquela página
